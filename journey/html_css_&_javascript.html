HTML, CSS & JavaScript

The first three things I was taught about on the Full Stack Developer course was HTML, CSS and JavaScript. I was only taught the basics of these three topics but many of the principles I learnt here have informed my learning from that point, e.g. the introduction of variables. By creating this website it has helped me to remember and build upon what I learnt here by putting it all into practice.

These three are key to building the front end of a website. So far while learning to code I have preferred the back end element of it, I have quite enjoyed learning ruby and the things you can create with it. At times while using HTML and CSS I have found it frustrating getting things how I want them to look, it is not just the case of dragging some boxes around the screen as some may think. While learning Ruby I have found it to be more intuitive to learn and write than JavaScript, however since creating this website I am starting to like the front end a bit more as this is what most people will see and appreciate.

HTML & CSS

Before learning anything about JavaScript I was taught about HTML & CSS. HTML is a markup language which is used to create web pages. HTML consists of various elements which can be used to organise and format a web page. For example when creating content for a web page you will use different elements or tags to represent headings and paragraphs, these are just two examples there are many more. Structuring websites in this way makes them accessible to everyone, it also allows other front end languages such as CSS and JavaScript to be used to make web pages more attractive and dynamic.

The introductory lessons for these two languages were pretty short but I have built on this knowledge a reasonable amount while creating this website. All the learning here was practical, the website freeCodeCamp was used where I worked through lots of short exercise in their browser based environment. FreeCodeCamp is definitely worth checking, another great resources which is free and covers a lot more subject, there is a Data Analysis with Python Course here which I wouldn't mind giving a go when I have a spare 300 hours. The final exercise to bring these two together was to try and recreate the Google home page. I didn’t find this exercise too difficult just my usual CSS dramas of trying to get objects to stay in place. You can see my attempt at this here.
